# Trivia App 🎯

Welcome to the **Trivia App**, a fun and interactive quiz application designed to test your knowledge! The app now features a **red-themed interface** for a bold and engaging look.

## 📖 Overview

Trivia App allows users to:
- Answer challenging trivia questions.
- Get immediate feedback on their answers.
- Improve their knowledge in a fun way.

The app is fully responsive and styled with a **red theme** for an immersive experience.

---

## ✨ Features

- 🛠️ **Custom Questions**: Answer a variety of preloaded trivia questions.
- 📱 **Responsive Design**: Works seamlessly on desktops, tablets, and smartphones.
- 🌟 **Real-Time Feedback**: Immediate indication of correct or incorrect answers.
- 🎨 **Red Theme**: Bold and vibrant design for an engaging quiz experience.

---

## 🖥️ Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/trivia-app.git
    cd trivia-app
    ```

2. Open the `index.html` file in your favorite web browser:
    ```bash
    open index.html
    ```

---

## 🚀 Usage

1. Launch the app in your browser.
2. Select a question and input your answer.
3. Submit your answer and receive instant feedback.
4. Enjoy and challenge yourself to improve your trivia skills!
